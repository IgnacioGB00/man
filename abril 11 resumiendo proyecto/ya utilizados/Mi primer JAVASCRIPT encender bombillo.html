Mi primer JavaScript

¿Qué puede hacer JavaScript en HTML?

JavaScript puede cambiar el contenido HTML.

¿Qué puede hacer JavaScript cambiando atributos HTML?

JavaScript puede cambiar los valores de los atributos HTML.

En este caso, JavaScript cambia el valor del atributo `src` (source) de una imagen.

Cambiar el estilo de un elemento

JavaScript puede cambiar el estilo de un elemento en HTML.

Ocultar un elemento

JavaScript puede ocultar elementos en HTML.
